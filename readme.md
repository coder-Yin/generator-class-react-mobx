#介绍
- react-mobx脚手架

- 仓库

    ```js
    npm i -g yo
    npm i generator-class-react-mobx -g
    yo /Users/mac/.nvm/versions/node/v6.9.0/lib/node_modules/generator-class-react-mobx
    ```

- 使用指南（步骤）
    - 输入期望生成的目标目录（字符串），支持以下几种：
        - 相对路径（相对当前命令行所在的目录）
        - 绝对路径
        - 不填（默认当前目录）
    - 输入期望生成的文件类型（选择）
        - 页面（默认）
        - 组件
    - 输入文件夹名称（字符串）
        - 默认aName
    - 输入store的名称（字符串）
        - 默认stroName
